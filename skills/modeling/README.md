discovery

## Finalidade

O diretório `modeling/` armazena conhecimento operacional reutilizável sobre modelagem de dados.

Este diretório é a fonte primária para capacidades relacionadas a modelagem.

A norma de maior precedência continua sendo:

- `../../MANIFEST.md`

---

## Dependências relacionadas

- `../../MANIFEST.md`
- `../README.md`

---

## Quando usar

Consulte `modeling/` quando precisar:

- estruturar entidades e contratos
- revisar coerência semântica
- orientar granularidade de dados
- melhorar clareza de modelagem

---

## Quando não usar

Não use `modeling/` como fonte primária para:

- regras normativas de modelagem
- governança estrutural
- definição de agente
- template de solicitação

Consulte, respectivamente:

- `../../rules/modeling/`
- `../../governance/`
- `../../agents/`
- `../../prompts/`
